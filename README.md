# nbtw

나비퀘스트추적기는 World of Warcraft Mists of Pandaria 5.4.8용 퀘스트 추적 애드온입니다.

- Addon folder: `nbtw`
- Interface: `50400`
- Slash command: `/nbtw`
- Author: `made by kimnabi88`

## 설치

1. `release/nbtw_MoP548.zip` 압축을 풉니다.
2. 압축 안의 `nbtw` 폴더를 WoW 5.4.8 클라이언트의 `Interface/AddOns/` 아래에 넣습니다.
3. 게임에서 애드온을 활성화하고 `/nbtw`로 창을 엽니다.

## 포함 내용

- `nbtw/`: 실제 애드온 폴더
- `nbtw/MistsOfPandaria/`: 판다리아 퀘스트 데이터
- `release/nbtw_MoP548.zip`: 배포용 ZIP

## 참고

이 저장소에는 작업용 원본 참고 폴더를 포함하지 않습니다. 업로드 대상은 실제 애드온 결과물과 릴리스 ZIP입니다.
